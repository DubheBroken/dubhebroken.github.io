Android开发环境搭建

2019-12-24 Android开发光速入坑

准备工作

开发安卓大致需要以下硬件配置：

4个框框以上 3GHZ 以上的 CPU（否则会卡到怀疑人生）

8G 以上的内存

60G 左右的固态硬盘空间（对，60G，一点都不夸张）

以下是可选项：

最好是 Intel 的 CPU

最好有一架梯子

下载Android Studio

Android Studio官方页面

看到那个大大的 Download 了吗？点他会弹出一个霸王条款，打勾然后点下面的按钮就可以下载了

（其实那个按钮就是 url，右键也可以下载）

这里下载下来的是 exe安装包，通常推荐使用免安装版本，所以下一步我们来

选择适合自己的版本

点上图左下角的 Download Options 然后你就会看到下面这张图

个人推荐使用zip包解压即用

测试版本和体验版本

体验版本（Canary）有最新的功能，更新最频繁，最快的时候3天一更，图标是黄色的

测试版本（Beta）是Canary迭代后的产物，比Canary更稳定，更新比Canary慢

正式版本（Stable）就是上面放在首页的版本，更新最慢，但最稳定，图标是绿色的

从顶部的 perview 可以跳转到Canary版本的下载链接，下图右边的是Canary，左边的是Beta

3个版本有单独的更新推送，插件通常可以互通，项目只要没有把 Gradle 更到Canary的测试版去，通常是不受影响的。也就是说你可以同时使用3个版本甚至更多版本的Android Studio

配置Android Studio

首次启动配置

选择配置目录
如果你曾经使用过其他版本的 Android Studio ，那么你首次启动的时候会弹出配置目录选择

如果你曾经使用的版本和当前版本的大版本号相同（比如3.5.1和3.5.2），选第一项就好了。

（其实大版本号变了也可以使用第一项，但是配置细则会被修改，导致同样使用这个目录的另一个Android Studio可能会无法正常加载配置）

如果你不知道选什么，那就选最后一个

注意：插件也在配置目录中，如果你要保留以前安装的插件，那就选第一项

配置代理

首次启动有可能会弹出这个，配置代理，如果你有的话，选第一项进去配置，没有就点 Cancel

代理配置

根据你的代理情况填写然后点 Check connection 校验，会弹出一个提示框，输入谷歌的首页，然后回车

如果看到这个框框就是配置成功了

首次启动配置

看图就好没啥好说的

选个皮肤

配置SDK

然后无脑Next、Finish即可，如果勾了模拟器会弹出 Intel 模拟器插件的安装界面，无脑next即可。

（AMD的CPU是装不上这个插件的，所以大可不勾模拟器）

下载SDK

见到这个界面就大功告成………

了一半了，嗯(⊙_⊙)

我们还有一堆SDK需要下载

下载SDK

点开这个 Configure ，常用的选项图上标注好了

选择 SDK Manager

SDK工具下载

展开详情后可以看到所有的版本，Build-Tools 官方推荐永远使用最新版本。

必备的组件会在首次安装时自动勾选，如果缺什么在你项目编译的时候会报出来的，如果看着烦可以统统勾选。

AVD镜像下载

AVD是 Android Vitrual Devices 的缩写，即 Android Studio 自带的模拟器，是目前在电脑上运行速度最快的安卓模拟器。

即使你不用模拟器，也一定要下载最新版本的镜像

展开详情以后会发现选项特别得多

其中4.4W是安卓手表

选项含义以7.1.1为例

勾选了要下载的项之后，点右下角的ok，会弹出提示框告诉你大概会占用多少硬盘空间，直接ok就好

有时候会弹出一个霸王条款界面，左侧选项目，右侧选同意，然后Next即可
如果进度条很慢很慢可以考虑搭个梯子，其实国内有源，只是不稳定，时快时慢。
强烈建议用梯子，因为 Gradle 也可以用到

环境配置完成

至此环境配置完成，可以准备 Hello World 了
